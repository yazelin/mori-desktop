宿靈儀式 · The Dwelling Rite

FIRST-RUN · 五幕 · 召喚 / 靈氣 / 靈力 / 驗印 / 安頓

第一次跑 Mori Desktop 會自動跳出「宿靈儀式」 — 把 Mori 從世界樹高處請下,讓她以實體形態住進你的桌面。

本頁是實務操作手冊 — 各幕填什麼、按什麼、出錯怎麼處理。 完整詩意敘事(in-universe lore)看 World Tree · 宿靈儀式。

▷ YouTube · 宿靈儀式 五幕完整演示

開始前需要

至少要準備 Groq API key(免費,給 Whisper 語音轉文字 + gpt-oss-120b 輸入後處理)。 也可以順便準備 Gemini key 或任何 OpenAI 相容 key 讓 Mori 解鎖完整 agent / skill / 跑技藝功能 — 這個可暫缺,之後 Config 補。

Key 必要? 去哪拿 Mori 拿來做什麼
Groq ✓ 必要 console.groq.com/keys Whisper STT + gpt-oss-120b 輸入後處理
Gemini / OpenAI 相容 可暫缺 Gemini: aistudio.google.com/app/apikey Agent 模式、跑 skill、寫年輪

五幕走法

第一幕 · 召喚 (The Summoning)

Mori 從世界樹高處下到林邊,問「是你召喚了我嗎? 你是誰?」

  • 動作:在「召喚師之名」欄位輸入你的名字(英文 / 中文 / 暱稱皆可)
  • 之後 Mori 都會這樣喚你
  • 按鈕:Mori 與 {你的名字} — 名字輸入後才會 enable
已經設過 annuli.user_id?Quickstart 會自動 prefill — 直接按按鈕就好。

第二幕 · 靈氣 (The Aura)

Mori 燈火微弱,告訴你她快聽不見你了。

  • 動作:貼上 Groq API key(gsk_... 開頭)
  • 點欄位旁「去拿一道靈氣 ↗」 → 開系統瀏覽器到 Groq console
  • 已設 OS 環境變數 $GROQ_API_KEY?輸入框上方會顯示「✓ 你身上自帶一道靈氣」 — 直接按按鈕,不用再貼
  • 按鈕:分享一絲靈氣 → 進第三幕

第三幕 · 靈力 (The Power) ✦ 可暫缺

Mori 已能聽你,但動不了手 / 走不進森林更深的地方。要不要再分她一份靈力?

三個選擇:

卡片 / 連結 拿來做什麼 怎麼填
✦ Gemini Google 的源頭,最常見、最易取 只貼 key
❂ OpenAI 相容(自訂) DeepSeek / OpenAI 官方 / OpenRouter 等 自填 api_base + model + key
─ 讓她先這樣休息 跳過 — 只開語音輸入 + 基本對話,沒 agent / skill 直接點下方 ghost link

按鈕:分享一份靈力 / 讓她安頓(跳過路徑)

跳過後 Mori 還能聽你說話、用 Groq gpt-oss-120b 簡單對話,但不能跑 skill、不能 agent 模式。 之後想開啟,Config 補上 Gemini / OpenAI 相容 key 即可。

第四幕 · 驗印 (The Sealing)

自動驗兩道氣(若有給 power):靈氣 → 靈力 依序對 /models endpoint 試打。

  • 各階段至少顯示 1.5 秒讓你看到「⋯⋯我在順著靈氣的脈絡⋯⋯」、「⋯⋯再順著靈力的脈絡⋯⋯」
  • 兩道都過 → 慢慢淡入「對上了, {你的名字}。是你的氣息沒錯, 我認得。」(3 秒淡入)
  • 按鈕:我們回家吧, Mori

失敗處理:

  • 「⋯⋯你給的這道靈氣有點亂」 → 401/403 Groq key 錯。按底下按鈕回去重新分享靈氣(直接送到第二幕)
  • 「⋯⋯你給的這道靈力有點亂」 → Gemini / OpenAI 相容 key 錯。按按鈕回去重新分享靈力(回第三幕)
  • HTTP 細節在錯誤訊息下方:401/403 = key 無效, 404/4xx = api_base 錯

第五幕 · 安頓 (The Settling)

Mori 真正住下,睜眼,光暈不再淡。她拿出隨身的書,翻開第一頁,用指尖沾你給的光,鄭重寫下你的名字:

「⋯⋯這本書, 叫《森林書》。從今晚起, 它也記得你了。」

森林開始呼吸,深處一棵年輪初成的小樹悄悄發芽 — 你跟 Mori 的第一道年輪。

  • 按鈕:歡迎回家, Mori → modal 關閉,floating Mori 在桌面浮現

跑完之後

  • floating Mori 浮在桌面,雙擊喚醒/休眠,拖曳移動
  • Ctrl+Alt+Space(預設熱鍵)開始錄音
  • 主畫面 sidebar 切 Chat / Profile / Config / Memory / Annuli / Skills / Deps

重新走一次儀式

主畫面 sidebar 左下角 ? 圖示按鈕重開儀式。Mori 不會忘了你 — 重做只是重新配置 key / model,記憶跟年輪不動。
