Sprite Sheet(5P 4×4 系統)

每個 state 對應一張 1024×1024 PNG-32 sprite sheet,內含 4×4 共 16 個 frame(每 frame 256×256),row-major 順序播放。 整套規格(manifest schema / loop mode / duration)集中在 character-pack.md。

整張 sheet
1024 × 1024 px
每 frame
256 × 256 px
grid
4 × 4(16 frame)
順序
row-major,左→右、上→下
loop_mode
idle / sleeping / recording / thinking = loop;done / error = one-shot
duration
manifest loop_durations_ms[state] — idle 3000、sleeping 5000、recording 1500、thinking 2000、done 600、error 800
實作位置
src/FloatingMori.tsx + src/floating.css(兩軸獨立 steps(4))

Poses & Actions

設計稿中有 8 種額外動作(讀書、提燈、撫貓、揮手、走路、思考等),目前 desktop 版 正式採用 idle / recording / thinking / done / error / sleeping 6 種(character-pack manifest 的 states 必含項)。Manifest 內 optional_states 已 預留 walking / dragging,未來擴充:

  • 📚 reading — 使用者打開「閱讀模式」(planned)
  • ✏️ writing — Mori 在用 RememberSkill / EditMemorySkill 時短暫切到此 pose
  • 🍵 chill — 長時間 idle 隨機切到此 pose 增加生氣(防呆板)

Items & Props

Mori 隨身道具,從設計稿提取:

提燈(lantern)

守護光 / 記憶之光。dock 主視窗開啟動畫可用作 motif。

書(book)

知識 / 記憶。Memory tab 的 icon 可往這方向走。

蕨葉(fern)

成長 / 共生。Mori sprite 髮飾 / 葉冠主要元素。

燈籠花(bell)

通知 / 喚醒。tray icon notification 狀態可用此意象。

Color Variations

設計稿底部有 Mori 的色彩變化版(夏 / 秋 / 冬 / 夜)— 給未來「節日 sprite」用。 目前 desktop 預設只 ship 森林綠主版;變化版未來會做成獨立 character pack (見 character-pack.md),從 Config tab → Appearance → Character picker 切換。

Voice Tone(角色說話風格)

跟 Brand 共用 — 詳見 brand.html § Voice Tone。重點:

  • 繁中、自然、不客套
  • 「我有 Jarvis,我有 Mori」— 個人森林精靈,不是公司 AI
  • 不會的事老實說「目前還沒這個能力」
