Tray

SYSTEM TRAY · ICONS · SPRITES · MENU

系統匣 icon 規範 + 7 個 sprite 表情 + menu icon 庫 + interaction states。
讓 Mori 在 panel 上是「會呼吸的森林精靈」,不是死木頭按鈕。

Design Reference

Tray Icon (App Identity)

系統匣的「app icon」永遠是 logo 圓徽,不是 sprite 表情。原因:

  • 小尺寸(16/22px)sprite 細節糊掉
  • logo 在所有狀態都不變 — 使用者「點 Mori」找它要穩定

檔案來源

  • docs/design/mori-logo.png — 從 brand sheet logo 區自動 crop + 去米色背景
  • crates/mori-tauri/icons/icon.png + 16/32/64/128/256 — Tauri bundle 輸出

單色版規則

系統匣有些桌面環境(GNOME / macOS dark panel)會自動把 icon 染成 monochrome。 要對應:

  • 主版用墨綠 #2C4A3D(淺底 panel)
  • 反白版用米色 #E6DECA(深底 panel)
  • 純黑 silhouette 也行 — GNOME-Shell 系統匣會自己處理 invert

Tray Menu(系統匣選單)

Right-click / left-click tray icon 出現的選單,目前實作在 main.rs:

項目動作
顯示 Morishow main window + 把 floating set_always_on_top
隱藏hide main window
對話模式Mode::Agent
語音輸入模式Mode::VoiceInput
休眠(關麥克風)Mode::Background
Voice Profile ▸掃 ~/.mori/voice_input/ 列出全部 .md(5K-2)
Agent Profile ▸掃 ~/.mori/agent/ 列出全部 .md
重新開始對話清空 conversation history
離開app.exit(0)

Notifications

Mori 主動通知時(背景排程提醒 / 失敗 / etc.):

  • tray icon 短暫變 notice 表情 0.6s 後恢復(planned, 5O-2)
  • 用桌面通知系統(tauri-plugin-notification)送原生 notif
  • Mori sprite 在 floating 同時呈現 attention pose(eyes wide / 望向 cursor)
